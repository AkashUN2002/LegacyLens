# Screenshots

Drop your LegacyLens screenshots in this folder. The main [README](../../README.md#screenshots) references these exact filenames, so keep the names as-is and the images will appear automatically.

| Filename | What to capture |
| --- | --- |
| `home.png` | The home screen — the `.zip` uploader and language selector, ideally mid-analysis showing the ingestion pipeline stages. |
| `chat.png` | The chat view with a grounded answer, its cited entities, and a confidence badge. |
| `graph.png` | The knowledge graph tab — nodes coloured by risk and sized by fan-in. |
| `compare.png` | The migration coverage report (legacy vs. modernized) with the headline metrics. |
| `roadmap.png` | The Deliverables tab — the interactive roadmap builder or the generated roadmap. |
| `observability.png` | The observability dashboard — token usage, cost, and latency. |

## Tips

- **Format:** PNG is preferred (crisp UI text). JPG/WebP work too, but update the extension in the README if you change it.
- **Resolution:** capture at a normal browser width (~1280–1600px wide) so text stays readable when scaled down.
- **Theme:** the app ships with a dark theme, so dark screenshots are expected.
- **Privacy:** avoid capturing real connection strings, API keys, or private source in the frame.
